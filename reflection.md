**Reflection on Learning Git and Using VS Code**

During the first phase of my coding course, I was introduced to Git and the fundamentals of version control. At first, the number of commands and concepts felt overwhelming. Learning about repositories, branches, commits, staging, merging, merge conflicts, and commit messages required me to develop a new way of thinking about how software projects are managed. However, as I practiced these skills in Visual Studio Code (VS Code), I began to appreciate how powerful and organized Git can be.

One of the most valuable lessons I learned was the importance of tracking changes and maintaining a history of my work. Using commands such as `git add`, `git commit`, and `git status` helped me understand the workflow developers use to save progress and document changes. Learning how to create branches and merge them back into the main project showed me how teams can work on different features simultaneously while keeping projects organized.

I also gained experience handling merge conflicts, which taught me the importance of carefully reviewing code and understanding how changes interact with one another. While these situations were challenging, they helped build my problem-solving skills and confidence.

Working in VS Code added another layer of learning. Navigating the terminal, remembering command sequences, and understanding the relationship between Git and the editor required patience and practice. Although it took time to become comfortable, each successful command and completed task increased my confidence.

What excites me most is that this has been the first area of technology that I genuinely want to stick with. Even when concepts are difficult, I find myself spending extra hours exploring, practicing, and experimenting. The combination of challenge and accomplishment has made learning Git both rewarding and enjoyable.

As our class moves forward into Python programming and NetAcad coursework, I am excited to continue building on this foundation. Learning Git has shown me that persistence and practice lead to growth, and it has strengthened my interest in pursuing technical skills and a future in technology.
